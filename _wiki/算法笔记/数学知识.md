# 数学知识

## 质数

若一个正整数无法被除了1和它自身之外的任何自然数整除，则称该数为质数(或素数)，否则称该正整数为合数。

在整个自然数集合中,质数的数量不多,分布比较稀疏,对于一个足够大的整数N,不超过N的质数大约有`N/lnN`个，即每`lnN`个数中大约有1个质数。

### 质数的判定

#### 试除法

只需要扫描2~$\sqrt{N}$之间的所有整数，依次检查它们能否整除N, 若都不能整除，则N是质数，否则N是合数。试除法的时间复杂度为$O(\sqrt{N})$。

```cpp
bool is_prime(int n){
    if (n < 2) return false;
    for (int i = 2; i <= sqrt(n); ++i)
        if (n % i == 0) return false;
    return true;
}
```

### 质数的筛选

#### Eratosthenes筛法

对于每个数$x$，我们只需要从$x^2$开始，把$x^2$， $(x + 1)\times{x}$， ${\lfloor{\frac{N}{x}}\rfloor}\times{x}$标记为
合数即可。

```cpp
void primes(int n){
    memset(v, 0, sizeof(v));
    for (int i = 2; i <= n; ++i){
        if (v[i]) continue;
        cout << i << endl; // i是质数
        for (int j = i; j <= n/i; ==j) v[i*j] = 1;
    }
}
```
