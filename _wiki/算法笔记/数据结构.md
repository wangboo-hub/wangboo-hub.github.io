# 数据结构

## 基础数据结构
### 二叉堆
二叉堆是一种支持插入、删除、查询最值的数据结构。它其实是一棵满 足“堆性质”
的完全二叉树”，树上的每个节点带有一个权值。

- 大根堆：树中的任意一个节点的权值都小于等于其父节点的权值。
- 小根堆：树中任意一个节点的权值都大于等于其父节点的权值。

根据完全二叉树的性质,我们可以采用层次序列存储方式,直接用一个数组来保存
二叉堆。层次序列存储方式，就是逐层从左到右为树中的节点依次编号,把此编号作为
节点在数组中存储的位置(下标)。在这种存储方式中，父节点编号等于子节点编号除
以2，左子节点编号等于父节点编号乘2,右子节点编号等于父节点编号乘2加1。

以小根堆为例

#### 初始化

```cpp
int heap[N], Size = 0;
```
#### 上移
##### 递归版
```cpp
void down(int x){
 int t = x;
 if(x*2 <= Size && heap[t] > heap[x*2])t = x*2;
 if((x*2+1) <= Size && heap[t] > heap[2*x+1])t = x*2+1;
 if(x != t){
  swap(heap[t], heap[x]);
  down(t);
 }
}
```
##### 迭代版

```cpp
void down(int x){
 int p = 2*x;
 while(p <= Size){
  if(p < Size && heap[p] > heap[p+1]) ++p;
  if(heap[p] < heap[x]){
   swap(heap[p], heap[x]);
   x = p; p *= 2;
  }else break;
 }
}
```
#### 下移

```cpp
void up(int x){
 while(x/2 && heap[x] < heap[x/2]){
  swap(heap[x/2], heap[x]);
  x>>=1;
 }
}
```
#### insert
`insert(val)`操作向二叉堆中插入一个带有权值`val`的新节点。我们把这个新节点直接放在存储二叉堆的数组末尾，然后通过交换的方式向上调整，直至满足堆性质。其时间复杂度为堆的深度，即`O(logN)`。
```cpp
void insert(int x){
    heap[++Size] = x; 
    up(Size);
}
```

#### 删除指定位置元素
`remove(p)`操作把存储在数组下标p位置的节点从二叉堆中删除。我们先把heap[p]与h`eap[Size]` 交换，然后令`Size`减小1。注意此时`heap[p]` 既有可能需要向下调整，也有可能需要向上调整，需要分别进行检查和处理。时间复杂度为`O(logN)`。

```cpp
void remove(int p){
 heap[p] = heap[Size--];
 up(p); down(p);
}
```

#### getTop
`getTop`操作返回二叉堆的堆顶权值，即最大值`heap[1]`， 复杂度为`O(1)`。

```cpp
int getTop(){
 return heap[1];
}
```

#### 删除堆顶元素

```cpp
void pop(){
 heap[1] = heap[Size--];
 down(1);
}
```

## 高级数据结构

### 二叉搜索树(Binary Search Tree)

#### BST的特征
- 每个元素有唯一的键值，这些键值能够比较大小。通常把这些键值存放到BST的节点上。
- 任意节点的键值，大于其左子树所有节点的键值，小于其右子树所有节点的键值。
- 在BST上，以任意节点为根节点的一棵子树仍然是BST。
- 键值最大的节点没有右孩子；键值最小的节点没有左孩子。

#### BST的建立

```cpp
int tot, root, INF = 1<<30;
struct BST{
    int l,r;
    int val;
}bst[SIZE];//数组模拟链表

int New(int val){
    bst[++tot].val = val;
    return tot;
}

void build(){
    New(-INF), New(INF);
    root = 1;
}
```

#### BST的检索
在BST中检索是否存在键值为val的节点。

设变量`p`等于根节点`root`，执行以下过程：
1.	若`p`的键值等于`val`，则已经找到。
2.	若`p`的键值大于`val`	
	+ 若`p`的左子节点为空，说明不存在`val`。
	+ 否则，在`p`的左子树中递归进行检索。
3.	若`p`的键值小于`val`	
	+ 若`p`的右子节点为空，则说明不存在`val`。
	+ 否则，在`p`的右子树中递归进行检索。

```cpp
int get(int p, int val){
    if (p == 0) return 0;
    if (val == bst[p].val) return p;
    return val < bst[p].val ? get(bst[p].l, val) : get(bst[p].r, val);
}
```

#### BST的插入
在BST中插入一个新的值`val`。

在发现要走向的p的子节点为空,说明`val`不存在时，直接建立关键码为`val`的新节点作为`p`的子节点。

```cpp
void insert(int &p, int val){
    if (p == 0){
        p = New(val);
        return;
    }
    if (val == 0) return;
    if (val < bst[p].val) insert(bst[p].l, val);
    else if (val > bst[p].val) insert(bst[p].r, val);
}
```
