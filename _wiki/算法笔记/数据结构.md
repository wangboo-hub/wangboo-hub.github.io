# 数据结构



## 高级数据结构

### 二叉搜索树(Binary Search Tree)

#### BST的特征
- 每个元素有唯一的键值，这些键值能够比较大小。通常把这些键值存放到BST的节点上。
- 任意节点的键值，大于其左子树所有节点的键值，小于其右子树所有节点的键值。
- 在BST上，以任意节点为根节点的一棵子树仍然是BST。
- 键值最大的节点没有右孩子；键值最小的节点没有左孩子。

#### BST的建立

```cpp
int tot, root, INF = 1<<30;
struct BST{
    int l,r;
    int val;
}bst[SIZE];//数组模拟链表

int New(int val){
    bst[++tot].val = val;
    return tot;
}

void build(){
    New(-INF), New(INF);
    root = 1;
}
```

#### BST的检索
在BST中检索是否存在键值为val的节点。

设变量p等于根节点root，执行以下过程：
1.	若p的键值等于val，则已经找到。
2.	若p的键值大于val	
(1).	若p的左子节点为空，说明不存在val。
(2).	否则，在p的左子树中递归进行检索。
3.	若p的键值小于val	
(1).	若p的右子节点为空，则说明不存在val。
(2).	否则，在p的右子树中递归进行检索。

```cpp
int get(int p, int val){
    if (p == 0) return 0;
    if (val == bst[p].val) return p;
    return val < bst[p].val ? get(bst[p].l, val) : get(bst[p].r, val);
}
```

#### BST的插入
在BST中插入一个新的值val。

在发现要走向的p的子节点为空,说明val不存在时，直接建立关键码为val的新节点作为p的子节点。

```cpp
void insert(int &p, int val){
    if (p == 0){
        p = New(val);
        return;
    }
    if (val == 0) return;
    if (val < bst[p].val) insert(bst[p].l, val);
    else if (val > bst[p].val) insert(bst[p].r, val);
}
```
