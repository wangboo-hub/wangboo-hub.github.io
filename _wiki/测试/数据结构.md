# 测试




## 二叉堆

以小根堆为例

#### 初始化

```cpp
int heap[N],Size=0;
```

#### 上移

##### 递归版

```cpp
void down(int x){
 int t=x;
 if(x*2<=Size&&heap[t]>heap[x*2])t=x*2;
 if((x*2+1)<=Size&&heap[t]>heap[2*x+1])t=x*+1;
 if(x!=t){
  swap(heap[t],heap[x]);
  down(t);
 }
}
```

##### 迭代版

```cpp
void down(int x){
 int p=2*x;
 while(p<=Size){
  if(p<Size&&heap[p]>heap[p+1])++p;
  if(heap[p]<heap[x]){
   swap(heap[p],heap[x]);
   x=p;p*=2;
  }else break;
 }
}
```

#### 下移

```cpp
void up(int x){
 while(x/2&&heap[x]<heap[x/2]){
  swap(heap[x/2],heap[x]);
  x>>=1;
 }
}
```

#### 插入元素

```cpp
void insert(int x){
 heap[++Size]=x;
 up(Size);
}
```

#### 删除指定位置元素

```cpp
void remove(int x){
 heap[x]=heap[Size--];
 up(x);down(x);
}
```

#### 获取堆顶元素

```cpp
int getTop(){
 return heap[1];
}
```

#### 删除堆顶元素

```cpp
void pop(){
 heap[1]=heap[Size--];
 down(1);
}
```

## STL  

### priority_queue

优先队列，默认是大根堆

##### 头文件

```cpp
#include<queue>
```

##### 操作

```cpp
//定义成小根堆的方式：
    priority_queue<int, vector<int>, greater<int>> q;
```

表达式 |意义
:--:|:--:
`size()`|返回容量大小
`empty()`|判断是否为空  
`push()`|插入一个元素
`top()`|返回堆顶元素
`pop()`|返回堆顶元素

### set, map, multiset, multimap

基于平衡二叉树（红黑树），动态维护有序序列
    size()
    empty()
    clear()
    begin()/end()
    ++, -- 返回前驱和后继，时间复杂度 O(log2n)

#### set/multiset

表达式 |意义
:--:|:--:
`insert()`|插入一个数
`find()`|查找一个数
`count()`|返回某一个数的个数
`erase()`|(1) 输入是一个数x，删除所有x   O(k + log2n)<br>(2) 输入一个迭代器，删除这个迭代器
`lower_bound(x)`|返回大于等于x的最小的数的迭代器
`upper_bound(x)`|返回大于x的最小的数的迭代器

#### map/multimap

`find()`|
`[]`|注意multimap不支持此操作。 时间复杂度是 O(log2n)
`lower_bound()/upper_bound()`|
